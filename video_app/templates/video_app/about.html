API Endpoints

RESTful API endpoints for integrating with external applications

Available API Endpoints

Use these endpoints to integrate gesture recognition into your applications

POST /upload/

Upload a video file for hand gesture recognition. Accepts raw video files (minimum 4 seconds) and returns recognized gesture text and video path.

POST /upload_text/

Convert text input to sign language video. Accepts text input and returns corresponding sign language video for the text.

POST /upload_voice/

Process voice notes and convert to sign language. Accepts audio files and returns transcribed text and sign language video.
